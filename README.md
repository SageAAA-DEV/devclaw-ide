# DevClaw IDE

**THE AI THAT ACTUALLY DOES THINGS**

DevClaw is an AI-powered coding environment built on VSCode. It integrates OpenClaw agents directly into your IDE, so you can build with AI that actually works where you work.

## Features

- **Native AI Integration** — OpenClaw agents built right into VSCode
- **Gateway Panel** — Monitor agents, skills, tools, sessions in real-time
- **Code Editor Tabs** — Full IDE with OpenClaw superpowers
- **Settings & Configuration** — Easy setup for your AI backend
- **Open Source** — Free to use, free to modify

## Getting Started

```bash
# Clone the repo
git clone https://github.com/bsci83/devclaw-oss.git
cd devclaw-oss

# Install dependencies
npm install

# Build
npm run compile

# Run
npm run dev
```

## Requirements

- Node.js 18+
- npm or yarn
